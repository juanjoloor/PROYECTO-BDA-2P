# PROYECTO-BDA-2P



## Documentación del sistema explicando el rediseño de la BDD
Se mantiene la estructura relacional distribuida. Esta estructura permite realizar consultas de manera eficiente ya que están relacionados con más de un conjunto de datos. Las lecturas se realizarían en relación a la claves primarias y foráneas de las tablas. Además, esta estructura permite que el sistema siga funcionando aún si sucede un error en algún nodo.
